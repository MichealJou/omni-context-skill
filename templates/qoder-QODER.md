# OmniContext For Qoder

OmniContext provides shared workspace context under `.omnicontext/`.

Required reading order:

1. `.omnicontext/INDEX.md`
2. `.omnicontext/projects/<project>/overview.md`
3. `.omnicontext/projects/<project>/handoff.md`
4. Relevant shared or personal files only if needed

Required update behavior:

1. Update `handoff.md` after meaningful work
2. Update `todo.md` when open tasks change
3. Update `decisions.md` when non-trivial choices are made

Never store secrets in `.omnicontext/`.
