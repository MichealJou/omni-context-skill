# References

默认语言：中文。

目录约定：

- `zh-CN/`：默认参考文档目录，完整可用
- `en/`：英文参考文档目录，完整可用
- `ja/`：日文参考文档目录，完整可用

默认加载规则：

1. 未明确指定语言时，优先读取 `references/zh-CN/`
2. 当用户、项目策略或工具入口明确要求英文时，读取 `references/en/`
3. 当用户、项目策略或工具入口明确要求日文时，读取 `references/ja/`
