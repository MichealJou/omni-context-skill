# Prompt Templates

These templates define the default operator-facing prompt style for English workflows.

## Default Usage

- Use this file only when the user, project policy, or tool configuration explicitly requires English

## English Prompt Template

```text
Use OmniContext to manage the current workspace.
Identify the target project first, then read the relevant overview, handoff, and index documents.
After completing the task, update handoff, todo, decisions, or document indexes when needed.
```
